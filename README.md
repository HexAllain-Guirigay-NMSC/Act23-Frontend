# GIS Frontend

## Requirements

Make sure you have installed:

* Node.js (v18 or higher)
* npm (comes with Node.js)

---

## Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/HexAllain-Guirigay-NMSC/Act23-Frontend.git
cd gis-frontend
```

---

### 2. Install dependencies

This will recreate the `node_modules` folder.

```bash
npm install
```

---


### 3. Run the project

```bash
npm run dev
```

Then open:

```bash
http://localhost:5173
```

---

## Notes

* Make sure the backend (Laravel API) is running
